# Smart Resume Screening

An AI-powered Resume Screening System built using Python and Flask. This application helps recruiters analyze resumes, extract skills, and rank candidates based on their similarity to job descriptions.

## Features

- Resume upload (PDF)
- Automatic text extraction
- Skill extraction
- Candidate ranking
- Similarity score calculation
- SQLite database for storing results
- Clean and user-friendly interface

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Scikit-learn
- PyPDF2

## Installation

```bash
git clone https://github.com/rishipandey28433-hash/Smart-Resume-Screening.git
cd Smart-Resume-Screening
pip install -r requirements.txt
python app.py
```
